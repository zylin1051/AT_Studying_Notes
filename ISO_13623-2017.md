# ISO 13623 --- The Design of Pipeline System for the Transportation of Fluids

## Note 

This page should be used as a refenrence only. **ALWAYS** refer the original document for details and clarity.

## Original Document

View the document (2017 version) [here](ISO_13623-2017.pdf).

## Scope

- Parent standards for pipeline design, construction, testing, opeation, maintenance, and abandonment
- Petroleum and natural gas (fluids) transportation
- Rigid, metallic pipelines (not application to flexible pipeline structures)
- On-land and offshore
- See **Figure 2 on Page 2** for the extent of pipeline systems covered

## Things to Consider when Referencing ISO 13623

| Question | Why it matters |
|---|---|
| Is this for petroleum/natural gas pipeline transportation? | If yes, ISO 13623 may be relevant. |
| Is it on-land or offshore? | Both can be covered. |
| Is it a rigid metallic pipeline? | If yes, it fits the material/type scope. |
| Is it a new system or a modification? | New systems are directly covered; modifications can be covered. |
| Is the item part of the pipeline system or only plant/process piping? | This decides whether ISO 13623 or another piping/process standard controls. |
| **Is the item pipe, valve, bend, flange, fitting, coating, CP equipment, or station equipment?** | **Different referenced standards apply (see Section 2).** |
| Is the reference dated or undated? | This determines which edition applies. |
| Does the client specification override or add to ISO 13623? | Project specs often add stricter requirements. |
| Are there local regulations? | ISO 13623 allows national requirements for safety/environment to apply. |
| Is the item within the pipeline system or inside a station/terminal? | This affects whether it is “pipeline” or “piping” under **Section 3**. |
