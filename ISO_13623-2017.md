## Scope

- Basis for pipeline design, construction, testing, opeation, maintenance, and abandonment
- Petroleum and natural gas (fluids) transportation
- Rigid, metallic pipelines (not application to flexible pipeline structures)
- On-land and offshore
- See **Figure 2 on Page 2** for the extent of pipeline systems covered


## Things to consider when using ISO 13623

| Question | Why it matters |
|---|---|
| Is this for petroleum/natural gas pipeline transportation? | If yes, ISO 13623 may be relevant. |
| Is it on-land or offshore? | Both can be covered. |
| Is it a rigid metallic pipeline? | If yes, it fits the material/type scope. |
| Is it a new system or a modification? | New systems are directly covered; modifications can be covered. |
| Is the item part of the pipeline system or only plant/process piping? | This decides whether ISO 13623 or another piping/process standard controls. |
| **Is the item pipe, valve, bend, flange, fitting, coating, CP equipment, or station equipment?** | **Different referenced standards apply (see Section 2).** |
| Is the reference dated or undated? | This determines which edition applies. |
| Does the client specification override or add to ISO 13623? | Project specs often add stricter requirements. |
| Are there local regulations? | ISO 13623 allows national requirements for safety/environment to apply. |
| Is the item within the pipeline system or inside a station/terminal? | This affects whether it is “pipeline” or “piping” under **Section 3**. |

## Section 5 - Pipeline System Design Basis
| Topic | Things to Consider |
|---|---|
| System boundary | Define where the pipeline system starts/ends and what interfaces are included. |
| Applicable requirements | Check project specs, client requirements, and local regulations in addition to ISO 13623. |
| Design conditions | Confirm design life, pressure, temperature, flow rate, fluid composition, and shut-in conditions. |
| Fluid category | Classify the transported fluid as Category A, B, C, D, or E. If uncertain, use the more conservative category. |
| Hydraulic analysis | Confirm steady-state and transient conditions have been considered. |
| Flow assurance | Check for hydrate, wax, slugging, erosion, high viscosity, or low-flow issues where applicable. |
| MAOP | Identify the maximum allowable operating pressure for the system or each section. |
| Pressure control | Confirm how normal operating pressure will be kept within MAOP. |
| Overpressure protection | Check whether relief, shutdown, or source-isolation protection is required. |
| Operation and maintenance | Consider operation, inspection, isolation, maintenance, repair, shutdown, and restart needs during design. |
| Corrosion management | Confirm coating, CP, monitoring, internal corrosion control, and inspection access are considered. |
| Pigging | Confirm whether pigging or in-line inspection is required and whether the system is compatible. |
| Public safety/environment | For on-land Category D/E fluids, check Annex B and environmental protection requirements. |
